Coder Platform Direct Menu

Instructions:

This page provides direct access to key features of the Coder Platform. After clicking a button:

  1. Return to the main application tab/window
  2. The requested UI element should appear
  3. If nothing happens, try clicking the button again

Manual Control via Console

// Open Projects Dashboard
localStorage.setItem('showProjectDashboard', 'true');
document.querySelector('.project-dashboard')?.style.display = 'flex';

// Open Main Menu
document.querySelector('.main-menu-container')?.style.display = 'block';
document.querySelector('.hamburger-button')?.click();

// Find and click New Project button
[...document.querySelectorAll('button')]
  .find(b => b.textContent.includes('New Project'))?.click();
